# 테스트 시나리오

## 1. 기능 테스트 시나리오

### 1.1 인증 기능

| ID | 시나리오 | 기대 결과 | 통과 |
|----|----------|-----------|------|
| AUTH-01 | 로그인 버튼 클릭 | OAuth 로그인 페이지로 이동 | ☐ |
| AUTH-02 | 로그인 성공 후 | 홈 페이지로 리다이렉트, 사용자 정보 표시 | ☐ |
| AUTH-03 | 로그아웃 버튼 클릭 | 세션 종료, 로그인 버튼 표시 | ☐ |
| AUTH-04 | 로그인 없이 보호된 페이지 접근 | 로그인 페이지로 리다이렉트 | ☐ |

### 1.2 게시글 CRUD

| ID | 시나리오 | 기대 결과 | 통과 |
|----|----------|-----------|------|
| POST-01 | 게시글 목록 페이지 접속 | 게시글 목록 표시, 최신순 정렬 | ☐ |
| POST-02 | 게시글 클릭 | 상세 페이지 이동, 내용 표시 | ☐ |
| POST-03 | 새 게시글 작성 (로그인 상태) | 게시글 저장, 목록에 표시 | ☐ |
| POST-04 | 게시글 수정 (작성자) | 수정 내용 저장, 반영 확인 | ☐ |
| POST-05 | 게시글 삭제 (작성자) | 삭제 확인 후 목록에서 제거 | ☐ |
| POST-06 | 빈 제목으로 작성 시도 | 유효성 오류 메시지 표시 | ☐ |

## 3. 에러 시나리오 테스트

### 3.1 네트워크 오류

| ID | 시나리오 | 기대 결과 | 통과 |
|----|----------|-----------|------|
| ERR-N01 | 오프라인 상태에서 목록 조회 | 에러 메시지 표시, 재시도 버튼 | ☐ |
| ERR-N02 | API 응답 지연 (5초+) | 로딩 인디케이터 표시 | ☐ |
| ERR-N03 | 서버 500 오류 | 사용자 친화적 에러 메시지 | ☐ |

### 3.2 입력 오류

| ID | 시나리오 | 기대 결과 | 통과 |
|----|----------|-----------|------|
| ERR-I01 | 제목 200자 초과 | 유효성 오류, 제한 안내 | ☐ |
| ERR-I02 | 본문 비어있음 | 유효성 오류, 필수 안내 | ☐ |
| ERR-I03 | 특수문자 입력 | XSS 방지, 정상 저장 | ☐ |

### 3.3 세션 오류

| ID | 시나리오 | 기대 결과 | 통과 |
|----|----------|-----------|------|
| ERR-S01 | 세션 만료 후 작성 시도 | 로그인 페이지로 이동, 안내 메시지 | ☐ |
| ERR-S02 | 다른 탭에서 로그아웃 후 작업 | 세션 상태 동기화, 재로그인 유도 | ☐ |
