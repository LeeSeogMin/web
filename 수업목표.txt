웹 프로그래밍(실전): AI 협업 기반 풀스택 리터러시
수업목표 (600자)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

본 과목은 AI 시대에 필요한 '바이브코딩(Vibe Coding)' 역량을 함양하는 웹 프로그래밍 실전 과정이다. 학생들은 HTML, CSS, JavaScript, React 등 웹 개발의 핵심 개념을 학습하고, GitHub Copilot과 Context7 같은 AI 코딩 도구를 활용하여 효율적으로 코드를 작성하는 방법을 익힌다. 단순히 코드를 암기하는 것이 아니라, AI와 협업하여 문제를 해결하고 빠르게 프로토타입을 구현하는 현대적 개발 방식을 체득한다. React, Supabase, Vercel을 활용한 풀스택 웹 애플리케이션을 직접 설계하고 배포하며, 인증, 데이터베이스, 권한 관리 등 실무 필수 기술을 습득한다. 최종적으로 학생들은 AI 도구를 전략적으로 활용하여 아이디어를 신속하게 구현할 수 있는 실전 웹 개발 능력을 갖추게 된다.

핵심 키워드:
• 바이브코딩 (AI 협업 기반 개발)
• React 프론트엔드
• Supabase 백엔드
• Vercel 배포
• GitHub Copilot/Context7
• 실전 풀스택 프로젝트
