# 제11장 집필계획서: Row Level Security (RLS)

## 1. 장 개요

### 1.1 위치 및 역할
- **Part 4**: Supabase 백엔드 통합의 네 번째 장 (마지막)

## 4. 핵심 용어

| 용어 | 정의 |
|------|------|
| RLS | Row Level Security, 행 단위 접근 제어 |
| Policy | 특정 조건을 만족하는 행만 접근 허용하는 규칙 |
| USING | SELECT, UPDATE, DELETE 시 조건 |
| WITH CHECK | INSERT, UPDATE 시 데이터 검증 조건 |
| auth.uid() | 현재 로그인한 사용자의 UUID |
| Service Role | RLS를 우회하는 관리자 권한 |
