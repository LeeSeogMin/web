# Chapter 1. 첫 배포 — create-next-app에서 Vercel까지 — 집필 계획서

## 장 개요
- **구성**: 개념 30% + 환경 설정 40% + 배포 실습 20% + 과제 10%
- **대상**: 학부생 (선수 지식 없음)
- **난이도**: 기초
- **핵심**: 첫 수업에서 Vercel 배포까지 완료

## 학습 목표
1. 웹 동작 원리(클라이언트-서버, HTTP)를 설명할 수 있다
2. Node.js, VS Code, Git 개발 환경을 구성할 수 있다
3. create-next-app으로 Next.js 프로젝트를 생성할 수 있다
4. Git으로 코드를 GitHub에 올릴 수 있다
5. Vercel을 통해 웹사이트를 배포할 수 있다

## 절 구성
| 절 | 내용 | 비고 |
|---|------|------|
| 1.1 | 웹이 동작하는 방식 | 클라이언트-서버, HTTP, 렌더링 (간결하게) |
| 1.2 | 개발 환경 설정 | Node.js, VS Code, Git — macOS/Windows 모두 |
| 1.3 | Next.js 프로젝트 생성 | create-next-app, 구조, dev 서버, DevTools |
| 1.4 | GitHub에 올리기 | git add/commit/push, 저장소 연결 |
| 1.5 | Vercel 배포 | 가입, 임포트, 자동 배포 |
| 1.6 | 과제 | 페이지 수정 → push → URL 제출 |
