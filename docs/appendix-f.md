## 버전 정보

이 교재에서 사용한 주요 라이브러리 버전:

| 라이브러리 | 버전 | 릴리스 |
|------------|------|--------|
| React | 18.2.x | 2022 |
| React Router | 6.x | 2021 |
| @supabase/supabase-js | 2.x | 2022 |
| Vite | 5.x | 2023 |
| Node.js | 18.x LTS | 2022 |

**주의**: 라이브러리는 지속적으로 업데이트된다. 공식 문서에서 최신 정보를 확인하고, Context7 등을 활용하여 버전 차이로 인한 문제를 해결한다.

## 문제 해결 자료

### 일반적인 오류

| 오류 | 참고 |
|------|------|
| CORS 에러 | MDN CORS 문서 |
| 401/403 인증 오류 | Supabase Auth 문서 |
| React hooks 규칙 위반 | React 공식 문서 Hooks 규칙 |
| 빌드 실패 | Vite 트러블슈팅 |

| 도구 | 용도 |
|------|------|
| React DevTools | React 컴포넌트 디버깅 |

| Supabase Dashboard | 데이터베이스, 로그 확인 |
| Vercel Logs | 배포 및 함수 로그 |
